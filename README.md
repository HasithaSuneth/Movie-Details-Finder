# Movie-Details-Finder
### Movies / TV series details finder software developed by using Python with PyQt5 module and OMDb API.

## Features
- Discover details of a <b>Movie</b> or <b>TV series</b> by searching by its <b>Name</b>.
<p align="center">
  <img width="500" src="https://user-images.githubusercontent.com/87106402/170817616-a3b85e3c-bd6a-442d-aa2e-a165479ea104.png" alt="Block Diagram">
</p>
<p align="center">

</p>

- Generate a name list of Movies located at the local drive, and discover details of those movies. (Path: Movie Details Finder v1.0\export\\)
  - Full Details (.JSON)
  - Found Movie List Details - Name, Categories & IMDB rating (.TXT)
  - Not Found Movie List - Name (.TXT)

## IMPORTANT
Users need to get their <b>API key</b> from <b>"http://www.omdbapi.com/apikey.aspx"</b> and place that key inside the <b>"API_Key.txt"</b> file located at the <b>"Movie Details Finder v1.0\data\\"</b> path.

## Technical Details
- Database - OMDb API (http://www.omdbapi.com/)
- Support operating systems (Tested) - Windows, Linux (Ubuntu)
- File Size - 38MB
